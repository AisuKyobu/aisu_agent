# Product

## Register

**Product** — this is an app UI / tool surface. An AI agent web console for developers. Design SERVES the product.

## Users & Purpose

- **Primary user**: Individual developers using the agent for coding, research, automation.
- **Context**: Desktop browser, seated, focused. Single-session task completion.
- **Job**: Ask the agent to complete a task (search, code, browse), get results, iterate.
- **Secondary use**: Public demo mode for job interviews — limited access, read-only tools.

## Brand & Personality

3 words: **precise, calm, modern**.

- Reference feel: Linear-style cleanliness + VS Code utility. A tool that stays out of the way.
- The UI should feel dense and capable, not decorative. Information density when needed, breathing room when not.
- **Anti-references**: SaaS marketing pages, glowing gradients, glassmorphism, dark-mode-by-default developer aesthetics that feel "hacker".

## Color Strategy

**Restrained** — tinted neutrals + one accent at ≤10% surface. The accent (pink/magenta) is the brand recall; nearly everything else is grayscale+tint.

## Accessibility

- WCAG AA minimum (4.5:1 body text, 3:1 large text)
- Respect `prefers-reduced-motion`
- 44px minimum touch targets
